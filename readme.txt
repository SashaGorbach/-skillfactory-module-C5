Телеграм бот t.me/BelarusWeather_TrialBot определяет погоду
белорусских городов, при вводе названия города. Если город отсутствует 
в базе, то бот нас об этом уведомит.
Для расширения списка городов необходимо в файл config.py
добавить название города и географическую широту и долготу города.

Данные о погоде берутся с сайта open-meteo.com
